# Changelog

## [2.3.0] - 2026-03-04

### 新增
- **断连自动重试**：AI 连接意外中断时自动重试最多 3 次（间隔 1s/2s/3s 递增），重试过程实时提示
- **错误反馈 Toast**：醒目的错误提示（SVG 图标 + 关闭按钮 + 8秒自动消失 + 左侧红色边条）
- **意外断连检测**：修复 Service Worker 休眠等场景下错误被静默吞掉的 bug

### 优化
- 所有 UI 图标统一为 SVG，移除 emoji
- 添加 `callbackFired` 锁防止回调重复触发

## [2.2.0] - 2026-03-04

### 修复
- **HTML 表格渲染**：修复 AI 报告中表格首行数据被错误渲染为标题样式的问题
- **表格断裂**：修复标题与表格内容之间有空行时表格断裂成两个的问题
- **停止按钮配色**：统一「停止生成」按钮为与「停止」按钮一致的勃艮第红配色

### 优化
- **投顾资格提示词**：在 LLM 通用背景和合规性分析提示词中加入主播持有投顾资格证的说明，避免合法荐股被误判为违规
- **Markdown 转 HTML 引擎**：重写表格解析逻辑，正确使用 separator 行区分 thead/tbody，修复 HTML 转义破坏表格标签的 bug
