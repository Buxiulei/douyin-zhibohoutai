# Changelog

## [2.4.0] - 2026-03-04

### 新增
- **分段报告生成**：AI 分析重构为三板块独立生成（合规性分析 / 直播框架分析 / 直播技巧优化）
- **一键生成综合报告**：自动按顺序生成三个板块并汇总，支持中途停止
- **打开报告**：生成完成后可直接在新标签页中预览 HTML 报告
- **双格式下载**：支持 HTML（精美排版）和 TXT（Markdown 原文）两种下载格式
- **Web Worker 后台保活**：标签页切到后台时采集和分析不被浏览器节流
- **四角缩放**：悬浮窗支持从四个角拖拽调整大小

### 修复
- **停止生成不生效**：background.js 添加 AbortController，port disconnect 时真正终止 HTTP 请求
- **停止回调链路**：llm.js 的 stop() 通过 _stopped 标志确保 onError 回调触发

### 优化
- 板块卡片默认折叠，整体收进「展开单项分析」，减少界面干扰
- 无数据时「一键生成综合报告」按钮自动 disabled
- 停止按钮改为全宽红色样式，与采集区停止按钮风格一致

## [2.3.0] - 2026-03-04

### 新增
- **断连自动重试**：AI 连接意外中断时自动重试最多 3 次（间隔 1s/2s/3s 递增），重试过程实时提示
- **错误反馈 Toast**：醒目的错误提示（SVG 图标 + 关闭按钮 + 8秒自动消失 + 左侧红色边条）
- **意外断连检测**：修复 Service Worker 休眠等场景下错误被静默吞掉的 bug

### 优化
- 所有 UI 图标统一为 SVG，移除 emoji
- 添加 `callbackFired` 锁防止回调重复触发

## [2.2.0] - 2026-03-04

### 修复
- **HTML 表格渲染**：修复 AI 报告中表格首行数据被错误渲染为标题样式的问题
- **表格断裂**：修复标题与表格内容之间有空行时表格断裂成两个的问题
- **停止按钮配色**：统一「停止生成」按钮为与「停止」按钮一致的勃艮第红配色

### 优化
- **投顾资格提示词**：在 LLM 通用背景和合规性分析提示词中加入主播持有投顾资格证的说明，避免合法荐股被误判为违规
- **Markdown 转 HTML 引擎**：重写表格解析逻辑，正确使用 separator 行区分 thead/tbody，修复 HTML 转义破坏表格标签的 bug
