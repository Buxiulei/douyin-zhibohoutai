# 抖音直播数据提取 — Chrome 插件

从抖音直播后台（`anchor.douyin.com`）自动提取**文字记录**和**直播间评论**数据。

## 安装

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的 **「开发者模式」**
3. 点击 **「加载已解压的扩展程序」**
4. 选择本文件夹 `抖音直播后台插件`

## 使用

1. 访问抖音直播后台页面 `https://anchor.douyin.com/anchor/review?type=0&roomId=...`
2. 页面右下角会出现 **「直播数据提取」** 悬浮窗
3. 悬浮窗可以拖拽移动，可最小化

### 采集数据

- 点击 **「▶ 开始采集」** 开始采集文字记录或评论
- 采集过程中自动滚动加载全部内容，进度条实时显示
- 可随时点击 **「⏹ 停止」** 中断采集

### 导出数据

- **📄 导出纯文本** — 仅内容，不含时间戳和发言人
- **📋 导出完整记录** — 包含时间戳、发言人和内容

导出的文件为 `.txt` 格式，自动下载到浏览器默认下载目录。

## 注意事项

- 插件仅在 `anchor.douyin.com` 域名下激活
- 采集过程中请勿手动滚动数据区域
- 采集时间取决于数据总量，大量数据可能需要数分钟
